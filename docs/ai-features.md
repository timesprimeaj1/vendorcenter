# AI Features Roadmap

## Current AI-Ready Foundations
- Structured domain entities for vendors, services, bookings, and reviews
- Location and category signals available for ranking
- Activity logs and analytics for behavior insights

## Planned AI Features
1. Personalized service recommendations
2. Smart vendor ranking by context (location, rating, reliability)
3. Natural-language search over services and providers
4. AI-assisted support triage and FAQ suggestions
5. Demand forecasting for categories and zones

## Data Required
- Search and click events
- Booking conversion events
- Vendor response and completion signals
- Review sentiment metadata

## Guardrails
- Privacy-aware event collection
- Role-based data access
- Human-review for high-impact admin actions
